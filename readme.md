Student Dropout Management System

A simple **Student Dropout Management System** developed using Object-Oriented Programming (OOP) in Python. This project is designed for beginners who have recently learned OOP concepts such as Classes, Objects, Constructors, Inheritance, and Method Overriding.


Features

 Add Student
 View All Students
 Search Student
 Update Student
 Delete Student
 Mark Student as Dropout
 View Active Students
 View Dropout Students
 Generate Student Report
 Simple Admin Login


Technologies Used

Python 3
Object-Oriented Programming (OOP)
